# 📚 Documentation Hub

Centralize knowledge for the Vibe Coding Infra project.

## ✨ Suggested Structure
- `architecture.md` — high-level diagrams and component overviews.
- `adr/` — Architecture Decision Records for system trade-offs.
- `runbooks/` — Operational guides for automation and deployment.
- `notes/` — Scratch space for experiments and future work.

Keep documents concise, versioned, and easy to follow.
