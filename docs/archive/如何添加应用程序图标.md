# 🎨 如何为"猫猫唱片公司"应用程序添加图标

## 📋 概述

macOS应用程序图标需要 `.icns` 格式，包含多种尺寸（16x16 到 1024x1024，包括Retina版本）。

---

## 🚀 方法1：使用现有图标文件（推荐）

如果您已经有一个 `.icns` 文件：

### 步骤1：准备图标文件

1. 确保您有一个 `.icns` 文件（例如：`AppIcon.icns`）
2. 将图标文件放到项目目录下（例如：`assets/icon/AppIcon.icns`）

### 步骤2：手动复制图标

```bash
# 创建Resources目录（如果不存在）
mkdir -p "猫猫唱片公司.app/Contents/Resources"

# 复制图标文件
cp assets/icon/AppIcon.icns "猫猫唱片公司.app/Contents/Resources/AppIcon.icns"

# 验证
ls -lh "猫猫唱片公司.app/Contents/Resources/"
```

### 步骤3：更新Info.plist（如果图标文件名不是AppIcon）

如果您的图标文件名不是 `AppIcon.icns`，需要修改 `Info.plist`：

```bash
# 编辑Info.plist
open "猫猫唱片公司.app/Contents/Info.plist"
```

添加或修改：
```xml
<key>CFBundleIconFile</key>
<string>您的图标文件名（不需要.icns扩展名）</string>
```

例如，如果图标文件是 `MaoMaoIcon.icns`：
```xml
<key>CFBundleIconFile</key>
<string>MaoMaoIcon</string>
```

## 🎨 方法2：从PNG创建.icns文件

如果您有PNG格式的图标：

### 步骤1：准备PNG图标

1. 准备一个高质量的PNG图标（建议1024x1024像素）
2. 确保图标是正方形

### 步骤2：使用图标生成工具

**方法A：使用在线工具**
1. 访问 https://cloudconvert.com/png-to-icns
2. 上传PNG文件
3. 下载生成的.icns文件

**方法B：使用iconutil（macOS自带）**

```bash
# 1. 创建iconset目录
mkdir -p AppIcon.iconset

# 2. 复制并重命名PNG图标到不同尺寸
# macOS需要以下尺寸（以1024x1024为源图）：
cp icon.png AppIcon.iconset/icon_16x16.png
sips -z 16 16 icon.png --out AppIcon.iconset/icon_16x16.png
sips -z 32 32 icon.png --out AppIcon.iconset/icon_16x16@2x.png
sips -z 32 32 icon.png --out AppIcon.iconset/icon_32x32.png
sips -z 64 64 icon.png --out AppIcon.iconset/icon_32x32@2x.png
sips -z 128 128 icon.png --out AppIcon.iconset/icon_128x128.png
sips -z 256 256 icon.png --out AppIcon.iconset/icon_128x128@2x.png
sips -z 256 256 icon.png --out AppIcon.iconset/icon_256x256.png
sips -z 512 512 icon.png --out AppIcon.iconset/icon_256x256@2x.png
sips -z 512 512 icon.png --out AppIcon.iconset/icon_512x512.png
sips -z 1024 1024 icon.png --out AppIcon.iconset/icon_512x512@2x.png

# 3. 转换为.icns
iconutil -c icns AppIcon.iconset -o AppIcon.icns

# 4. 复制到应用程序包
mkdir -p "猫猫唱片公司.app/Contents/Resources"
cp AppIcon.icns "猫猫唱片公司.app/Contents/Resources/AppIcon.icns"

# 5. 清理临时文件
rm -rf AppIcon.iconset
```

### 简化脚本（一键转换）

创建 `scripts/create_icon_from_png.sh`：

```bash
#!/bin/bash
# 从PNG创建.icns图标

PNG_FILE="$1"
ICONSET_DIR="AppIcon.iconset"
ICNS_FILE="AppIcon.icns"

if [ -z "$PNG_FILE" ] || [ ! -f "$PNG_FILE" ]; then
    echo "用法: $0 <png文件路径>"
    exit 1
fi

echo "🎨 从PNG创建图标..."
mkdir -p "$ICONSET_DIR"

# 生成所有尺寸
sips -z 16 16 "$PNG_FILE" --out "$ICONSET_DIR/icon_16x16.png"
sips -z 32 32 "$PNG_FILE" --out "$ICONSET_DIR/icon_16x16@2x.png"
sips -z 32 32 "$PNG_FILE" --out "$ICONSET_DIR/icon_32x32.png"
sips -z 64 64 "$PNG_FILE" --out "$ICONSET_DIR/icon_32x32@2x.png"
sips -z 128 128 "$PNG_FILE" --out "$ICONSET_DIR/icon_128x128.png"
sips -z 256 256 "$PNG_FILE" --out "$ICONSET_DIR/icon_128x128@2x.png"
sips -z 256 256 "$PNG_FILE" --out "$ICONSET_DIR/icon_256x256.png"
sips -z 512 512 "$PNG_FILE" --out "$ICONSET_DIR/icon_256x256@2x.png"
sips -z 512 512 "$PNG_FILE" --out "$ICONSET_DIR/icon_512x512.png"
sips -z 1024 1024 "$PNG_FILE" --out "$ICONSET_DIR/icon_512x512@2x.png"

# 转换为.icns
iconutil -c icns "$ICONSET_DIR" -o "$ICNS_FILE"

# 复制到应用程序包
APP_NAME="猫猫唱片公司"
if [ -d "${APP_NAME}.app" ]; then
    mkdir -p "${APP_NAME}.app/Contents/Resources"
    cp "$ICNS_FILE" "${APP_NAME}.app/Contents/Resources/AppIcon.icns"
    echo "✅ 图标已添加到应用程序包"
else
    echo "⚠️  应用程序包不存在，请先运行: make create-app"
fi

# 清理
rm -rf "$ICONSET_DIR"

echo "✅ 图标创建完成: $ICNS_FILE"
```

使用方法：
```bash
chmod +x scripts/create_icon_from_png.sh
./scripts/create_icon_from_png.sh your_icon.png
```

## 🔄 刷新图标缓存

添加图标后，可能需要刷新macOS的图标缓存：

```bash
# 方法1：重新启动Finder
killall Finder

# 方法2：清理图标缓存
sudo rm -rf /Library/Caches/com.apple.iconservices.store
killall Finder
```

然后：
1. 删除应用程序（如果已安装）
2. 重新安装或直接使用应用程序包

---

## 📝 注意事项

1. **图标文件名**：通常使用 `AppIcon.icns`，也可以在Info.plist中指定其他名称
2. **文件大小**：高质量的.icns文件可能几MB
3. **图标尺寸**：建议源图至少1024x1024像素
4. **图标设计**：
   - 使用正方形图标
   - 考虑圆角效果（macOS会自动添加）
   - 确保小尺寸时也清晰可见
